# Standard TreasuryThis is a repo for managing machine readable artifacts for the [Standard Treasury APIs](http://www.standardtreasury.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Standard Treasury APIs](http://www.standardtreasury.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.standardtreasury.com/)- [Developer](http://www.standardtreasury.com/)- [Twitter](https://twitter.com/StandardAPI)- [GitHub](https://github.com/standardtreasury)The [APIs.json](https://github.com/api-evangelist/standard-treasury/blob/master/apis.json) index provides links to all the relevant links for the Standard Treasury APIs, including the OpenAPI and Postman if they are present.
